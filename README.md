# CyberDefenders Writeups

This repository contains my writeups for CyberDefenders Blue Team, SOC Analyst, and Digital Forensics challenges.

The goal of this repository is to document my investigation methodology, findings, and lessons learned while solving real-world security scenarios involving network forensics, incident response, malware analysis, threat hunting, and web attack investigations.
